# Full Stack Projects

A collection of full-stack web applications built using modern technologies such as Node.js, Express.js, MongoDB, JavaScript, HTML, and CSS.

Each project is contained in its own folder and includes detailed documentation, setup instructions, and source code.

## Projects

### 1. [OTP Verification System](./otp-verification-system/)

A full-stack OTP authentication that generates and sends a 4-digit OTP to a user's email address and verifies it using MongoDB and Nodemailer.

**Tech Stack:**

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* MongoDB
* Nodemailer
